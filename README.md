# Nexus-AE
A programmable logic matrix and generative sequencer for the AE Modular synth format.

The clectric Nexus shrinks an entire experimental CMOS logic synthesizer down to a AE-1U (25mm) footprint. Using a sleek 8x8 "blueprint" OLED matrix, you can route internal oscillators through high-speed logic gates (AND, OR, XOR, etc.) to create massive drone chords, rhythmic glitches, and metallic ring-modulation.

But the Nexus isn't just a static virtual patchbay. By feeding Control Voltage into the X and Y inputs, you can physically sweep a "ghost cursor" across to connect internal patches. This turns the matrix into a living sequence, scanning through different logic states and rhythmic subdivisions on the fly.

The weird wild fun comes when you enable the "Ouroboros" rows at the bottom and start feeding the signals back into themselves. 

# Key Features
- 8x8 Logic Matrix: Route any row (Sources) to any column (Logic Gates) using dual encoders.
- The "Ghost" Cursor: Use 0-5V CV to dynamically scan the X and Y axes, turning static logic into generative sequences.
- Internal Engine: Built-in 1V/Oct tracking oscillators mean it makes sound right out of the box.
- Performance Memory: Save and instantly recall up to 8 distinct matrix patches via internal EEPROM.
- Utility Hub: 4 Ins, 4 Outs, and integrated passive MULTs to maximize your AE Modular rack space.
